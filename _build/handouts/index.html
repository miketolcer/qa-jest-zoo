Unit Testing & Selecting Elements

Introduction

In this lab, you’ll be completing 3 different exercises. The first two give you the chance to work with Jest and writing unit tests. The third will have you working with HTML and CSS to find selectors for UI elements.

Setup

  1. Download the lab materials from Frodo

  2. cd into the lab directory

  3. Run npm i to install the necessary packages

npm run test

Throughout the lab, you will be using npm run test to run your tests.

Part One

  1. In the JestPractice folder, open the math.test.js file

  2. Complete the methods inside the operations object

  3. Write unit tests for the operations in the Math Tests block

  4. Run your tests to see if they pass

  5. Next move to the palindrome.test.js file

  6. Use assertions to test the pallyCheck function

  7. Run your tests

Part Two

  1. Open up the animal.js and zoo.test.ts files in the ZooLab folder

  2. Read over the Animal class and figure out the constructor and feed methods

  3. In your test file, test each of the foods with the feed function to make sure each one works as expected

Part Three

Setup

  1. Create a file in this lab’s directory called selectors.txt, keep track off all your findings in that file

Google

  1. Go to Google

  2. Find 2 selectors for the search bar

  3. Find what type of HTML elements are the “Google Search” and “I’m Feeling Lucky” buttons

  4. Find a selector for the Google logo

Amazon

  1. Visit Amazon

  2. Find a selectr for the login button

  3. Find a selector for the search bar

Enter Wanted

  1. Go to the Enter Wanted app

  2. Find a selector for each of the input fields on the site

Push your work to GitHub
